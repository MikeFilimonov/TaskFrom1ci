Designed to store information about countries, unions, and regions. The catalog data is used to specify country for the contact address and country of origin of some goods.

Open it with the Countries command of the Catalogs section. The list can store only several countries necessary for work.

Fill it in manually or using the Classifier of countries of the World. Open the Classifier using the relevant command.

The list displays:

  • The Name and Full name fields respectively store the short and full official name of a country. To present a country, e.g. for print, the short name will be used by default. If a country does not have a full name in the list, the full name and the short names are equal.
  • The Code field contains the three-digit numeric code of a country to accurately identify it according to the Classifier. For countries, unions and other data not related to the Classifier (e.g. European Union), it is recommended to enter "--".
  • The Alfa-3 code and Alfa-3 code fields contain a double-digit  (alfa-2) and a three-digit (alfa-3) codes for identification of states, dependencies, and principal administrative entities within states according to the standard ISO 3166.

For better convenience, you can add countries to the list from the Classifier as necessary. When necessary, you can edit a country if its data has been changed and the ID has not  been updated yet.

Entering manually

Sometimes, you might need to enter a region or a community, e.g. European Union.

  • To add a new country to the country list, click Create. 
  • It is not recommended to manually enter countries available in the Classifier due to a higher probability of error.
  • You can select a country from the Classifier. To confirm your selection, click Yes.
  • To enter a country manually, e.g. "European Union", click No. Fill in the required fields. Instead of numeric code, "--" is suggested automatically, since this value is used in print forms for the countries not present in the Classifier
