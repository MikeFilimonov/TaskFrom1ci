Designed to manage the report options, configure the placement in reports panels of the application sections.

You can save and use multiple options of the same report. Predetermined report options are available. You can create your own report options by editing the settings of the existing ones: generation and appearance settings (field content, sorting, grouping, conditional appearance).

You can also create common report options. Common report options are available for all the users of this report.

You can navigate from the application sections to the reports panels available in these sections. If there are a lot of reports, they are distributed among the section groups. To make your reports panels more convenient, configure them, e.g. hide the rarely used report options and setup quick access to the most popular panels.

The report options administrators have extra options for configuring the reports panel appearance: they can mark a report option as important and put it into the See also group.

Open the list with the All reports command in the reports panel of the application section. The sections and groups in this list depend on the available permissions and roles. Only the application administrator has full access to all the report options.

The application administrator can open the list by clicking the Report options command in Administration - Print forms, reports and processors. Some commands or actions available for the administrator can be unavailable in the All reports list. These features are mentioned in the list capacities description.

The list consists of two parts:

  • In the left part, there is a tree of command interface sections and groups having reports panels.
    • As you navigate between the tree nodes, the list is automatically filtered by a selected section or group.
    • Use the Including subordinate check box to show all the reports of a section or a group, including the report options in the nested groups. 
  • The reports list is in the right.

The list displays:

  • Name - a full report option name that will be shown in the lists.
  • Author - a full name of the author of this report option.
  • Description - detailed information about the report option.

To add columns to the list, click More - Edit form (the settings are saved):

  • Image Available to all the users:
    • Shown - the report option is available to all users.
    • Hidden - the report option is available only to the author.
  • Technical information:
  • Report name - a report registration name.
  • Report - a full report name (representation).
  • Report type - there are three report types:
    • Internal
    • Additional
    • External - options of external reports can not be located in the reports panel.
  • Option key - a name (unique ID) of the predetermined user report option. 
  • Parameters and filter names - a list of report parameters and filters.
  • Field names - a list of the report field names.

    Search

    If the report options list is long, use search to find a necessary report option quickly.

    • In the Search field, enter a necessary value, and then click Search. The selected section (group) will be searched.

      • You may search reports by multiple words, a word, or a part of word. Search is carried out:

        • In report names
        • In report option descriptions
        • In representations of the report option fields
        • In representations of parameters and filters of the report option
        • In names of user settings of the report option
        • In names of the sections and groups of the application and reports panel.

    Generating report

    • Select a report option, click Open, customize necessary settings, and then click Generate.
    • You can also double-click a necessary report option in the All reports list.

    Customizing report option

    • Select a report option, click Edit, and then edit necessary fields.
    • In the Report options list designed only for the administrator, double-click a necessary report option.

    Deleting report option

    • To mark a user report option for deletion, in the More menu, click Mark for deletion or press Del.
    • Repeat this command to clear Mark for deletion. 
    • Predetermined report options do not have this option.

    Placing report options in application sections 

    The report option may be placed by the author or the administrator.

    • You can drag report options from the current section/group. Hold Ctrl while dragging the report option to add it to a new group and keep it in the current group. The application prompts for confirmation of the report option placement. Click Yes to confirm.

    • Use the More - Place in section command to configure the placement of multiple report options: specify the command interface sections and groups where they will be placed.

    • You can use extra features for configuring the report options appearance: mark a report option as Important or move it to See also.

    •  In the administrator's Report options list, you can put the Place in sections command to the list command panel for better convenience.

    Resetting settings

    • Click More - Reset placement settings to reset the initial settings of the selected report options: placement in the application sections, default access and visible. The command can be run only for predetermined report options.
    • Click More - Reset user settings to reset the user settings of the selected report options: visible and quick access.
