Displays report options in the application sections or section groups.

You can open the report panel by running the relevant command from the application section.

Use the report panel to work with reports in a single window and group report options by command interface groups. You can work with the report panel in two modes:

  • View mode
  • Setup mode

View mode

  • Opened for the first time, the reports panel prompts you to enable or disable report description. You can also do it in the setup mode.
  • The report panel groups correspond with the application section groups containing the report.
    • The report panel can have other grouping types depending on a specific report list.
    • The See also group usually contains rarely used report options. 
    • The names of important reports in the report panel are typed in bold. 
    • You can customize these groups in the all reports list (use the All reports command to open it).
  • Click Setup or press F2 to go to the report panel setup mode.

Report search

The search function is available both in view mode and in setup mode.

  • You can find reports by a word (part of a word) or multiple words. Search covers:
    • Report name
    • Report option description
    • Report option field presentations
    • Parameter presentations and report option filters
    • Report option user setting names
    • Report panel/application section/group names
  • In the Search field, enter a necessary value and click Find. By default, you search in the current section. Select check boxes to specify where to search:
    • In this section only (by default)
    • In all sections (only in view mode)
      • Search results are presented under the Reports found in other sections title; the found reports are grouped according to application sections; each title is a link to the report panel of another section.
  • Your search settings and search terms are remembered.

Generating report

  • Click the report name.

Resetting report panel settings

  • If reports have been placed incorrectly, click More - Reset my settings to return to the default report panel settings.

All reports

  • Click All reports to go to the all reports list. IN the list, you can customize report options and their placement in all the application sections.

Setup mode

  • To go to this mode, click Settings (or press F2).
  • To set up display of report option descriptions, use the Report description check box.
  • Use check boxes to control option visible in the group.
  • Use the context menu to move a report option to quick access or open it to edit its name, description or placement settings:
    • To use quick access, right-click a report option in the list and select the Move to quick access command.
    • To cancel quick access to an option, right-click it and select Remove from quick access command.
    • To open a report option and edit its name, description and placement settings, right-click the report option and select the Edit command.
  • Click Settings (press F2) to save changes and go to the report panel view mode.
