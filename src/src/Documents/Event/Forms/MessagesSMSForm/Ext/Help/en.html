SMS messaging

You can send SMS messages directly from the application. To do this, you should have an agreement with one of the supported communication service providers and specify the identification data on the Settings – Personal organizer – SMS sending properties menu:

Specify the provider, login and password according to the agreement. You can use the hyperlinks at the bottom to go to the websites of the provider and find detailed description of services. Once you adjusted the settings, you can send the SMS messages. You can open the Events list in the main sections, for example, go to the Sales – see also – Events, and create the event of the SMS message kind.

The event appearance of SMS message is similar to the Email event. Specify the list of recipients in the header, type the message. You can use the message subject only for quick content filling or mailing lists thematic partitioning and can not include it into the message. The Delivery status field is next to the Phone numbers column. It is filled automatically and indicates the state SMS message sent to a certain recipient. The statuses are updated with data received from the communication service provider by pressing the Update delivery statuses button.

You can select the Convert Cyrillic into Latin check box in the lower part of the form. If this check box is selected, the text is automatically converted into "translit" text when you send the SMS. This feature can be useful as the maximum size of one SMS message depends on the language. For Latin characters the default message size is 140 characters, for the Cyrillic ones – 70 characters. You should keep in mind that the part of the characters can be reserved, for example, when sending the SMS messages chain. The estimated number of messages in a chain and the remaining characters are displayed at the right bottom. The exact number depends on the provider, so you need to contact your communication service provider. Fill the Sender name field if you want to indicate the name of the recipient instead of his phone number. You can change the default settings for entering the sender name in the Settings – Personal settings – Email and SMS section.
