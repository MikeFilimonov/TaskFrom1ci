Goods return documents

There are various reasons why your customers may wish to return merchandise they bought from your company. These might include the poor quality of the merchandise, a mistaken purchase of the wrong product, defective items, or extra items the customer ordered. The same reasons may force you to return merchandise your company bought from your supplier.

To register the title to the goods transfer for the goods your customer returns to your company, you register a goods return document in 1C:Drive. On the contrary, if your company returns goods to your supplier, you issue a goods return to supplier document to the supplier in 1C:Drive.

In 1C:Drive, you use separate goods return documents in addition to credit note documents if the Sales return flow option is set to "Credit note document posts accounts receivable, Goods return posts inventory entries."

Similarly, you use separate goods return to supplier documents in addition to debit note documents if the Purchase return flow option is set to "Debit note document posts accounts payable, Goods return posts inventory entries."

Viewing goods return documents

To view the list of goods return documents:

  • Click Sales > Return > Goods return.

Viewing goods return to supplier documents

To view the list of goods return to supplier documents:

  • Click Purchases > Return > Goods return.

Goods return fields

A goods return has the following fields:

  • Counterparty (required). The customer you register the goods return from.
  • Contract (required). The customer's contract that governs the issue of the goods return.
  • Sales invoice (required). The sales invoice you want to apply the goods return to.
  • Credit note. The credit note you want to apply the goods return to.
  • Number. A unique goods return number.
  • Dated. The date and time when your register the goods return.
  • Incoming number. A goods return number the customer assigned. Typically, it is the number of the delivery note.
  • Dated. The date and time when the customer issued the goods return.
  • Company (required). Your company that registers the goods return from the customer.
  • Warehouse (required). Your company's warehouse to which you return the goods.

Goods return to supplier fields

A goods return to supplier has the following fields:

  • Counterparty (required). The supplier you issue the goods return to.
  • Contract (required). The supplier's contract that governs the issue of the goods return.
  • Supplier invoice (required). The supplier invoice you want to apply the goods return to.
  • Debit note. The debit note you want to apply the goods return to.
  • Number. A unique goods return number. It helps you identify the goods return.
  • Dated. The date and time when your issue the goods return.
  • Company (required). Your company that issues the goods return to the supplier.
  • Warehouse (required). Your company's warehouse from which you return the goods.
  • GL account (required). The GL account to which you register the goods return. Typically, it is the purchase return GL account.

Creating goods return documents

To create a goods return document:

  1. Do any of the following:
    • Click Sales > Return > Goods return and then click Create.
    • Open a sales invoice or a credit note and click Generate > Goods return.
  2. Fill the goods return fields.

    For more information, see Goods return fields.

  3. Click the Products tab.
  4. Do any of the following:
    • To populate the table on the Products tab according to the sales invoice you specified in the Sales invoice field, click Fill by sales invoice.
    • To populate the table on the Products tab according to the credit note you specified in the Credit note field, click Fill by credit note.
  5. For each returned product, in the Return quantity column, specify the returned quantity.
  6. Click Post.

Creating goods return to supplier documents

To create a goods return to supplier document:

  1. Do any of the following:
    • Click Purchases > Return > Goods return and then click Create.
    • Open a supplier invoice or a debit note and click Generate > Goods return.
  2. Fill the goods return to supplier fields.

    For more information, see Goods return to supplier fields.

  3. Click the Products tab.
  4. Do any of the following:
    • To populate the table on the Products tab according to the supplier invoice you specified in the Supplier invoice field, click Fill by supplier invoice.
    • To populate the table on the Products tab according to the debit note you specified in the Debit note field, click Fill by debit note.
  5. For each returned product, in the Return quantity column, specify the returned quantity.
  6. Click Post.
