Designed to select report options.

Open it from the report option form with the More - Select option command.

You can select from:

  • Standard predefined report options. By default, it is common and shown in the list with character .
  • Common report options that other users opened for common access. IN the list, they are displayed with character  . You can choose options of all the authors.
  • Pre-saved personal report options.

The selection list may contain three report option groups:

  • Report options in use (not separated as a group, located at the top of the list).
  • Hidden in report panels - this group consists of predefined common reports with a cleared By default shown in report panel checkbox.
  • Marked for deletion - this group is always at the bottom of the selection list. When you select an option marked for deletion, you see a warning message.

The list shows:

  • Name - a full name of the report option shown in the lists.
  • The Available for all users  image:
    • Shown - all users can work with this report option.
    • Hidden - only the author can work with this report option.
  • Author - a full name of the user who created the report option.
  • At the bottom, you see the Description, detailed information about the selected report option.

Selecting report option

  • To open a report option, select it in the list and click Select.
  • Otherwise, double-click it.

Filtering

  • If the report option list is too long, you can use the filter to select a necessary option.
    • In the Author field, select the author of the report option from the users list.

Customizing report option

  • Select a report option, click More - Edit, and then edit necessary fields.

 Deleting report option

  • You can mark for deletion a user report option with the command of the More menu. You can also press Del.
  • To clear deletion mark, run this command again. 
  • You can not use this option with predefined report options.

To administrator/user responsible for reports

  • Usually personal report options of other authors are hidden. But you may occasionally need to make a personal report from this list available to all users or assign it to another user. To view all the report options, select the Show personal report options of other authors check box in the More menu.
