Designed to store a report option.

When a report is customized and generated, you can store it in a new report option. To do it, click Report options - Save options or More - Save option.

List of existing options

  • When saving a report option, you see the list of the existing predefined options and user options of this report.
    • The list includes:
      • Standard, predefined report options. By default, they are common.
      • Common report options opened for common access by other users.
      • Personal report options saved before.
    • The list also shows the report options marked for deletion.
  • You can Edit the report options existing in the list with the relevant command in the context menu. For a predefined report option, you can edit the description or location settings. For a user option, you can edit the name, too (if you have enough rights).
  • To delete an existing user report option, select it and press Del or use the Mark for deletion command of the context menu. You can not delete predefined report options.

Naming report option

  • To save your new report option, enter a Name not occupied by another option of this report. The saving procedure checks that all the report names are unique.
  • When you overwrite an option, you can save the settings in the existing option: by default, you select the report option for which the Save option command is executed. To overwrite, select any personal report option from the list. The report options that can not be overwritten (including the predefined options), automatically get "-copy" to their names. If the replaced report option was marked for deletion, you see a warning. Click Yes to confirm your choice.
    • The administrators and the users responsible for reports can overwrite reports by other authors.

Availability

  • The report option administrator can select value Only for author in the Available option box to limit the accessibility of a report option.
  • If you select accessibility for All users, you can allow its visible for other performers. To show the report option directly in the report panels for all users, select the Shown in report panel by default check box.

Saving report option

Use one of the following methods to save a report option:

  • When the basic parameters are defined, click Save. Thus, you will use the default settings and put the new report option to the same section of the reports list as the original option.
  • Click Next to edit the location settings for the report option. 

Placing in report panels

  • Select one or more check boxes in the Section, group column to place the report option in the relevant application sections or groups. When placed in a section, the report option will be displayed at the top of the report panel without placing in the groups.
  • In the Importance column, select the method to select an option in the group:
    • Important – the report option will appear bold.
    • See also – the report option will be displayed in a separate group at the bottom of the report panel.
  • Enter a Description that will be shown under the report name in the report panel and in the report lists.
  • Click Save to save the report option with the selected settings.
